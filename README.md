# Slowly Enhance — 信件统计助手

> 一款为 [Slowly Web](https://web.slowly.app/) 打造的 Chrome 浏览器扩展，帮助你统计与好友往来的信件数量、文字数、图片及语音等数据。

---

## 项目简介

**Slowly** 是一款模拟传统书信往来的社交应用。**Slowly Enhance** 通过拦截 Slowly Web 端的网络请求，自动收集并存储信件数据，让你能够以可视化的方式查看与每位好友的通信统计。

---

## 功能特性

- 📊 **总览统计**：一键查看总信件数、发出 / 收到数量、总字数、图片数、语音数
- 👥 **好友分组**：区分正常好友、隐藏好友与已删除好友
- 🏆 **好友排行**：按信件数量对所有好友进行排名
- 📝 **逐人统计**：查看与单个好友往来的详细数据，包括：
  - 发 / 收信数量
  - 中文字数 & 英文单词数
  - 图片与语音附件数量
  - 平均回复时间（我方 & 对方）
  - 首封 / 最近一封信日期及通信天数
- 📅 **活动热力图**：按日历展示历史收发信件的活跃程度
- 📈 **字数趋势图**：直观呈现信件字数随时间的变化趋势
- 🌍 **国家 / 地区统计**：统计好友来源及信件寄出地的地区分布
- 🏷️ **邮票统计**：记录使用过的各种邮票及使用次数
- ⚡ **快速翻页**：在 Slowly 网页内提供悬浮面板，支持自动滚动加载更多信件，加速数据收集

## 安装说明

本扩展为未打包扩展（开发者模式），需手动加载，暂未上架 Chrome 应用商店。

1. 下载或克隆本仓库到本地：
   ```bash
   git clone https://github.com/He-Songg/slowly_enhance.git
   ```

2. 打开 Chrome 浏览器，在地址栏输入：
   ```
   chrome://extensions/
   ```

3. 开启右上角的 **开发者模式（Developer mode）**。

4. 点击 **加载已解压的扩展程序（Load unpacked）**，选择克隆下来的项目文件夹。

5. 扩展安装完成后，图标将出现在 Chrome 工具栏中。

## 使用说明

1. 打开 [https://web.slowly.app/](https://web.slowly.app/) 并登录你的账号。

2. **收集好友列表**：正常浏览好友列表页面，扩展会自动拦截并保存好友数据。

3. **收集信件数据**：点击进入与好友的对话页面，逐页浏览信件（或使用页面右下角浮动面板中的「快速翻页」按钮自动滚动），扩展会自动收集每一封信件的内容与附件信息。

4. **查看统计**：点击 Chrome 工具栏中的 **Slowly Enhance** 图标，弹出窗口会显示总览数据及好友排行。

5. **查看详细统计**：点击弹出窗口底部的「查看详细统计」按钮，进入独立统计页面，可切换标签页查看热力图、趋势图、邮票统计、国家统计等。

6. **清除数据**：点击弹出窗口底部的「清除数据」按钮，可删除本地所有已收集的数据。

> **提示**：数据全部存储在浏览器本地的 IndexedDB 中，不会上传至任何服务器。

## 项目结构

```
slowly_enhance/
├── manifest.json          # 扩展配置文件（Manifest V3）
├── background.js          # Service Worker，负责打开标签页等后台任务
├── inject.js              # 注入到页面主世界（MAIN world），拦截 fetch / XHR 网络请求
├── content.js             # Content Script 主入口，处理数据计算与统计逻辑
├── popup.html             # 扩展弹出窗口 UI
├── popup.js               # 弹出窗口交互逻辑
├── stats.html             # 详细统计页面 UI
├── stats.js               # 详细统计页面交互与图表逻辑
├── icons/                 # 扩展图标
└── utils/
    ├── data-store.js      # IndexedDB 数据存储模块（好友 & 信件 & 元信息）
    ├── interceptor.js     # Content Script 端网络数据接收与解析
    └── page-helper.js     # 页面悬浮指示器与快速翻页辅助功能
```

## 数据说明

| 数据类型 | 存储位置 | 说明 |
|----------|----------|------|
| 好友信息 | IndexedDB `friends` 表 | 包含正常、隐藏、已删除好友 |
| 信件内容 | IndexedDB `letters` 表 | 包含正文、附件分类、时间戳、发件人等 |
| 元信息   | IndexedDB `meta` 表    | 包含当前登录用户 ID 等 |

附件分类规则：
- **图片**：`.jpg` `.jpeg` `.png` `.gif` `.webp` `.bmp` `.svg` `.heic` 等格式
- **语音**：`.mp3` `.m4a` `.aac` `.ogg` `.wav` `.opus` `.flac` `.amr` `.3gp` 等格式

## 已知问题

> ⚠️ **语音消息的数量目前无法统计。**
>
> 由于 Slowly API 返回的附件信息中，语音文件的识别依赖文件扩展名，部分语音消息可能不包含可识别的扩展名，导致统计结果不准确或偏低。

## 权限说明

本扩展仅申请以下权限：

| 权限 | 用途 |
|------|------|
| `storage` | 用于 Chrome 扩展 API 存储（当前主要使用 IndexedDB）|
| `host_permissions: https://web.slowly.app/*` | 在 Slowly Web 页面注入脚本、拦截数据 |
| `host_permissions: https://api.getslowly.com/*` | 拦截 Slowly API 响应以收集数据 |

## 兼容性

- 需要 **Chrome 88+**（Manifest V3 支持）
- 仅支持 **网页版 Slowly**（[https://web.slowly.app/](https://web.slowly.app/)）
- 不支持 Slowly 移动端 App
