# Slowly Enhance — 信件统计助手

> 一款为 [Slowly Web](https://web.slowly.app/) 打造的 Chrome 浏览器扩展，帮助你统计与好友往来的信件数量、文字数、图片及语音等数据。

---

## 项目简介

**Slowly** 是一款模拟传统书信往来的社交应用。**Slowly Enhance** 通过拦截 Slowly Web 端的网络请求，自动收集并存储信件数据，让你能够以可视化的方式查看与每位好友的通信统计。

## 功能特性

- 📊 **总览统计**：一键查看总信件数、发出 / 收到数量、总字数、图片数、语音数
- 👥 **好友分组**：区分正常好友、隐藏好友与已删除好友
- 🏆 **好友排行**：支持排序切换（信件数 / 字数 / 图片数 / 最近联系）
- 📝 **逐人统计**：查看与单个好友往来的详细数据，包括：
  - 发 / 收信数量
  - 中文字数 & 英文单词数
  - 图片与语音附件数量
  - 平均回复时间（我方 & 对方）
  - 首封 / 最近一封信日期及通信天数
- 📅 **活动热力图**：按日历展示历史收发信件的活跃程度（类似 GitHub 贡献图）
- 📈 **字数趋势图**：直观呈现信件字数随时间的变化趋势
- ⏱️ **回信速度趋势**：按月展示双方回信速度变化曲线
- 🕐 **活跃时段分析**：24 小时并排双柱状图，展示你和对方通常在什么时段写信，带时段分区背景（深夜 / 上午 / 下午 / 晚上）和悬浮数值提示
- 🌍 **国家 / 地区统计**：统计好友来源及信件寄出地的地区分布
- 🏷️ **邮票统计**：记录使用过的各种邮票及使用次数，展示 Slowly 官方邮票缩略图，自动获取邮票英文名称与描述信息
- 🎨 **自定义配色**：内置 4 套主题配色（蓝紫、暖橙、青绿、玫瑰），选择后自动保存，下次打开自动应用
- 💾 **数据备份与导出**：
  - **全局导出**：
    - **JSON** — 完整数据备份，包含所有好友和信件原始数据
    - **HTML 存档** — 精美排版的信件存档页面，按好友分组，可直接在浏览器中查看
    - **TXT** — 纯文本格式，按好友分组，每封信以分隔线隔开
    - **CSV** — 好友统计数据表格
    - **年度总结 HTML** — 一键生成指定年份的年度总结页面（总览/按月柱状/Top 好友等）
  - **单好友导出**：在好友详情页可单独导出该好友的数据
    - **JSON** — 该好友的所有信件数据
    - **HTML** — 排版精美的信件存档
    - **TXT** — 纯文本格式，每封信以分隔线隔开
- 📝 **词频统计（词云）**：在好友详情页展示 Canvas 绘制的词云图，支持「全部 / 我的 / 对方」三标签切换，词语大小按频率加权，水平与垂直混排，鼠标悬浮显示出现次数
- 📸 **长图导出**：总览页和好友详情页均支持一键截取整页长图，导出为 PNG 文件
- ✏️ **编辑器工具栏**：在 Slowly 网页编辑器中自动浮现格式化工具栏，支持一键缩进（8 空格段首）、全文缩进、合并空行、去除尾部空格、一键格式化等；支持快捷键：Tab 缩进 / Shift+Tab 反缩进 / Ctrl+Shift+F 一键整理 / Ctrl+Shift+L 清理空行
- ⚡ **快速翻页**：在 Slowly 网页内提供悬浮面板，支持自动滚动加载更多信件，加速数据收集
- 🧩 **体验小优化**：统计页卡片支持折叠/展开并记住状态；popup/统计页提供 GitHub 主页快捷跳转

## 安装说明

本扩展为未打包扩展（开发者模式），需手动加载，暂未上架 Chrome 应用商店。

1. 下载或克隆本仓库到本地：
   ```bash
   git clone https://github.com/He-Songg/slowly_enhance.git
   ```

2. 打开 Chrome 浏览器，在地址栏输入：
   ```
   chrome://extensions/
   ```

3. 开启右上角的 **开发者模式（Developer mode）**。

4. 点击 **加载已解压的扩展程序（Load unpacked）**，选择克隆下来的项目文件夹。

5. 扩展安装完成后，图标将出现在 Chrome 工具栏中。

## 使用说明

1. 打开 [https://web.slowly.app/](https://web.slowly.app/) 并登录你的账号。

2. **收集好友列表**：正常浏览好友列表页面，扩展会自动拦截并保存好友数据。

3. **收集信件数据**：点击进入与好友的对话页面，逐页浏览信件（或使用页面右下角浮动面板中的「快速翻页」按钮自动滚动），扩展会自动收集每一封信件的内容与附件信息。

4. **查看统计**：点击 Chrome 工具栏中的 **Slowly Enhance** 图标，弹出窗口会显示总览数据及好友排行。

5. **查看详细统计**：点击弹出窗口底部的「查看详细统计」按钮，进入独立统计页面，可切换标签页查看热力图、趋势图、邮票统计、国家统计等。

6. **切换主题**：在详细统计页面顶栏右侧点击彩色圆点即可切换配色方案（不包含暗黑模式）。

7. **导出好友数据**：在好友详情页底部点击导出按钮，可将该好友的信件单独导出为 JSON / HTML / TXT。

8. **清除数据**：点击弹出窗口底部的「清除数据」按钮，可删除本地所有已收集的数据。

> **提示**：数据全部存储在浏览器本地的 IndexedDB 中，不会上传至任何服务器。

## 项目结构

```
slowly_enhance/
├── manifest.json          # 扩展配置文件（Manifest V3）
├── background.js          # Service Worker，负责打开标签页等后台任务
├── inject.js              # 注入到页面主世界（MAIN world），拦截 fetch / XHR 网络请求
├── content.js             # Content Script 主入口，处理数据计算与统计逻辑
├── popup.html             # 扩展弹出窗口 UI
├── popup.js               # 弹出窗口交互逻辑
├── stats.html             # 详细统计页面 UI（支持 CSS 变量主题切换）
├── stats.js               # 详细统计页面交互与图表逻辑（含词云 Canvas 绘制）
├── icons/                 # 扩展图标
├── lib/
│   └── html2canvas.min.js # 页面截图库（长图导出）
└── utils/
    ├── data-store.js      # IndexedDB 数据存储模块（好友 & 信件 & 邮票元数据 & 元信息）
    ├── interceptor.js     # Content Script 端网络数据接收与解析（含邮票 API 拦截）
    └── page-helper.js     # 页面悬浮指示器、快速翻页辅助与编辑器工具栏
```

## 数据说明

| 数据类型 | 存储位置 | 说明 |
|----------|----------|------|
| 好友信息 | IndexedDB `friends` 表 | 包含正常、隐藏、已删除好友 |
| 信件内容 | IndexedDB `letters` 表 | 包含正文、附件分类、时间戳、发件人等 |
| 邮票元数据 | IndexedDB `stampMeta` 表 | 缓存邮票的英文名称、描述、国家等信息 |
| 元信息   | IndexedDB `meta` 表    | 包含当前登录用户 ID 等 |
| 主题偏好 | `chrome.storage.local` | 用户选择的配色方案 |

采集与性能说明：
- **增量采集**：已存在的信件不会重复写入，后续补抓时更快、更省资源。

附件分类规则：
- **图片**：`.jpg` `.jpeg` `.png` `.gif` `.webp` `.bmp` `.svg` `.heic` 等格式
- **语音**：`.mp3` `.m4a` `.aac` `.ogg` `.wav` `.opus` `.flac` `.amr` `.3gp` 等格式

## 已知问题

> ⚠️ **语音消息的数量统计仍可能不完全准确。**
>
> 由于 Slowly API 返回的附件信息中，语音文件的识别依赖多种字段与内部类型编码，部分语音消息可能不包含可识别的特征，导致统计结果存在一定偏差。

本扩展仅申请以下权限：

| 权限 | 用途 |
|------|------|
| `storage` | 用于保存主题配色偏好及 Chrome 扩展 API 存储 |
| `host_permissions: https://web.slowly.app/*` | 在 Slowly Web 页面注入脚本、拦截数据 |
| `host_permissions: https://api.getslowly.com/*` | 拦截 Slowly API 响应以收集数据 |

## 兼容性

- 需要 **Chrome 88+**（Manifest V3 支持）
- 仅支持 **网页版 Slowly**（[https://web.slowly.app/](https://web.slowly.app/)）
